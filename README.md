\# pod-flipkart — Proof of Delivery (POD) Prototype



\*\*Author:\*\* Jyoti  

\*\*Project:\*\* Flipkart Interview Assignment — Logistics Proof of Delivery  

\*\*Stack:\*\* React (frontend) • Node.js + Express (backend) • MongoDB (local) • Multer (file upload)





\## What this project does

A simple prototype for Proof of Delivery (POD):



1\. \*\*Enter AWB\*\* number (manual input; QR scanning possible)  

2\. \*\*Capture photo/video proof\*\* using a camera or select a file  

3\. \*\*Upload proof\*\* to backend (`server/uploads/`)  

4\. \*\*Store metadata\*\* (AWB, file path, timestamp) in MongoDB
